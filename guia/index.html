Back-End

O desenvolvimento back-end é a espinha dorsal de qualquer aplicação web moderna. Responsável por
gerenciar bancos de dados, autenticação de usuários, regras de negócio e a comunicação entre o front-end
e o servidor, o back-end garante que tudo funcione de forma segura, eficiente e escalável. Este guia foi
criado para ajudar você a entender os principais conceitos, ferramentas e boas práticas dessa área essencial
do desenvolvimento web, servindo tanto para iniciantes quanto para quem já possui alguma experiência
e deseja aprofundar seus conhecimentos.

Por: Ana da Silva Cardoso

Figma

Este guia completo para aprender Figma é ideal para quem deseja dominar uma das ferramentas de design
de interfaces mais utilizadas no mercado. Nele, você vai entender desde os conceitos básicos, como a
navegação pela interface e uso das principais ferramentas, até recursos avançados como criação de
componentes, prototipagem interativa e colaboração em tempo real. Ao longo do conteúdo, são abordadas
boas práticas de design, dicas para otimizar seu fluxo de trabalho e atividades práticas que ajudam a fixar o
aprendizado. Seja você iniciante ou alguém que já tem alguma experiência, este guia oferece uma jornada
clara, prática e eficiente para transformar suas ideias em interfaces digitais profissionais.

Front-End

Este guia para aprender Front-End foi criado para quem quer dar os primeiros passos no desenvolvimento
web e também para quem deseja consolidar seus conhecimentos. Ele aborda de forma clara e prática os principais
pilares da área: HTML, CSS e JavaScript, além de introduzir conceitos modernos como responsividade, acessibilidade,
consumo de APIs, versionamento com Git e uso de frameworks como React. Ao longo do material, você aprenderá a estruturar
páginas, estilizar elementos, criar interações dinâmicas e desenvolver projetos reais que simulam os desafios do mercado,
construindo uma base sólida para sua jornada como desenvolvedor Front-End.

Por: Joana Almeida Santiago
